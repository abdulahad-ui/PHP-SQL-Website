NOVANEST STORE SETUP

1. Extract this folder into htdocs or www.
2. Start Apache and MySQL in XAMPP or WAMP.
3. Open phpMyAdmin and import database.sql.
4. Check db.php and update MySQL password if needed.
5. Open http://localhost/NovaNest_Store/index.php
